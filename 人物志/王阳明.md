# 王阳明（王守仁）

> **生卒年**：1472-1529  
> **字**：伯安，**号**：阳明  
> **时代**：明代  
> **领域**：哲学、军事、文学、教育  
> **称号**：心学集大成者、王阳明、阳明先生

---

## 📜 生平大事记

### 早年时期（1472-1499）

- **1472 年**：出生于浙江余姚，父王华系状元，官至南京吏部尚书
- **5 岁**：仍不会说话但已默记祖父所读之书，改名后开口讲话
- **12-17 岁**：读私塾，17 岁遇道士学习养生术
- **18 岁**：遇娄谅学习"格物致知"，遍读朱熹著作
- **经典典故**：**守仁格竹** — 为实践"格物"格竹七天七夜，一无所获而病倒，从此怀疑宋儒学说

### 科举与仕途（1492-1506）

- **1492 年**：浙江乡试中举
- **1499 年**：中进士，授刑部主事
- **1504 年**：主持山东乡试
- **1506 年**：因上书营救戴铣等人，触怒宦官刘瑾 → **杖四十，贬贵州龙场驿丞**

### 龙场悟道（1508-1510）

- **贵州龙场**：万山丛薄，苗僚杂居，环境艰苦
- **核心领悟**：**"圣人之道，吾性自足，向之求理于事物者误也"**
- **撰写**：《教条示龙场诸生》 — 史称"**龙场悟道**"

### 南赣平乱（1516-1518）

任都察院左佥都御史，巡抚南赣汀漳：
- 连破**40 余寨**，斩杀俘获**7000 余人**
- 设立崇义县
- 推行《十家牌法》《南赣乡约》
- 兴办书院，教化畲民

### 平定宁王之乱（1519）

**鄱阳湖之战经典战役**：
- 朱宸濠发动叛乱
- 王守仁直捣南昌，迫使朱宸濠回援
- 鄱阳湖决战三天，生擒宁王
- **叛乱 35 天平定**

### 晚年讲学（1521-1529）

- **1521 年**：明世宗即位，封**新建伯**
- **1524 年**：在绍兴创建阳明书院，传播"王学"
- **1529 年**：病逝于江西，享年**57 岁**

## 🧠 核心思想体系

### 心学三大基石

1. **心即理**
   - 主张**"心外无理，心外无物"**
   - 理不在外物，而在心中
   - "心之所发便是意，意之本体便是知"

2. **知行合一**
   - **"知是行之始，行是知之成"**
   - 知与行不是两个步骤，而是一个过程
   - "未有知而不行者，知而不行，只是未知"

3. **致良知**
   - **良知**：人内心与生俱来的道德判断能力
   - **致良知**：将内心良知推广到事事物物
   - 晚年提炼的核心思想

### 四句教

```
无善无恶心之体
有善有恶意之动
知善知恶是良知
为善去恶是格物
```

### 与朱熹理学对比

| 维度 | 朱熹理学 | 王阳明心学 |
|------| -------- | --------- |
| 理的位置 | 理在事物中 | 理在心中 |
| 求知路径 | 格物致知 | 发明本心 |
| 知与行 | 先知后行 | **知行合一** |
| 修养方法 | 研究外物 | **内省良知** |

## ⚔️ 军事政治实践

### 军事成就
- **"狡诈专兵"**：用兵诡诈，独断专行
- **南赣平乱**（1516-1518）
- **平定宁王之乱**（1519）：35 天平叛，以少胜多

### 政治实践
- **设置新县**：平和县、崇义县、和平县
- **教化民众**：对畲民的汉化影响深远
- **基层制度**：恢复秦汉唐保伍制度
- **教育改革**：创建书院，提高教师待遇

## 🌏 历史影响

### 对中国
- 明代中后期思想解放的重要推手
- 泰州学派等后学的理论源头
- 明末清初启蒙思想的重要源头

### 对日本（阳明学）
- 江户时代的"阳明学"运动
- **明治维新的精神动力之一**
- 影响人物：西乡隆盛、吉田松阴等
- 口号："**尊王攘夷**"与阳明心学密切相关

### 对韩国
- 朝鲜时代的实学运动
- 对李滉（退溪）等哲学家影响深远
