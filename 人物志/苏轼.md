# 苏轼（苏轼/苏东坡）

> **生卒年**：1037-1101  
> **字**：子瞻，**号**：东坡居士  
> **时代**：北宋  
> **领域**：文学、书法、绘画、政治  
> **称号**：唐宋八大家之一、东坡居士

---

## 📜 生平大事记

### 早年时期（1037-1057）

**出生背景**
- 1037 年出生于四川眉山
- 父亲：苏洵（著名文学家）
- 母亲：程氏（知书达理）
- **天资聪颖**：年少即显露文学才华

**少年成名**
- 1056 年（20 岁）：携弟苏辙随父进京赶考
- 1057 年（21 岁）：进士及第，文章轰动京师
- 主考官欧阳修极加赞赏，叹为奇才
- 母亲去世，回籍守制

### 仕途起伏（1057-1079）

**早期仕宦**
- 1061 年：授大理寺评事，签书凤翔府判官
- 1066 年：父丧，回籍守制
- 1069 年：重返京师，任殿中丞、直史馆

**与王安石变法**
- 反对王安石激进变法，主张渐进改革
- 因政见不合，外放杭州通判
- 历任密州、徐州、湖州知州

**乌台诗案**
- 1079 年：被人指控以诗歌诽谤朝廷
- 被捕入御史台狱（乌台为御史台别称）
- 几乎被处死，后经多方营救得免
- 贬为黄州团练副使

### 黄州时期（1080-1084）

**人生转折**
- 1080 年：43 岁开始黄州贬谪生活
- 自号"东坡居士"（自开垦东坡荒地而得名）
- **创作高峰**：写下《念奴娇·赤壁怀古》《前赤壁赋》《后赤壁赋》等千古名篇
- 躬耕东坡，自食其力，写下"东坡肉"食谱

**生活态度**
- 面对逆境豁达乐观
- "一蓑烟雨任平生，也无风雨也无晴"
- 与佛印等僧人交往密切

### 重返朝廷（1085-1093）

**司马光执政**
- 1085 年：宋哲宗即位，高太后听政
- 司马光废新法，苏轼复起
- 官至礼部尚书、侍读

**两难处境**
- 既反对新法弊端，也反对全盘废弃新法
- 主张客观评价，各取所长
- 因此得罪新旧两党，再遭外放

### 晚年流放（1094-1101）

**再度贬谪**
- 1094 年：新党重新掌权
- 被贬惠州（今广东惠州）
- 1097 年：再贬儋州（今海南岛，当时为极僻远之地）

**海外生活**
- 在儋州三年，办学堂，教当地人读书
- "我本海南民，寄生西蜀州"
- 1100 年：宋徽宗即位，大赦天下

**最后的岁月**
- 1101 年：北返途中病逝于常州
- 享年 64 岁
- 临终前言："着力即差"

## 🎨 文学成就

### 诗词作品

**词作成就**
- 开创豪放词派，打破艳词传统
- 代表作：
  - 《念奴娇·赤壁怀古》"大江东去，浪淘尽，千古风流人物"
  - 《水调歌头》"明月几时有，把酒问青天"
  - 《江城子》"十年生死两茫茫"（悼亡妻千古绝唱）
  - 《定风波》"莫听穿林打叶声"

**诗歌成就**
- 题材广泛，风格多样
- 善用白描，语言清新
- 哲理与情感并重
- 代表作：
  - 《题西林壁》"横看成岭侧成峰，远近高低各不同"
  - 《饮湖上初晴后雨》"欲把西湖比西子，淡妆浓抹总相宜"

**散文成就**
- 唐宋八大家之一
- 文风豪迈流畅，善用比喻
- 代表作：
  - 《前赤壁赋》《后赤壁赋》
  - 《石钟山记》
  - 《记承天寺夜游》
  - 各种奏议、书信、题跋

### 书法绘画

**书法**
- "宋四家"之首（苏轼、黄庭坚、米芾、蔡襄）
- 擅长行书、楷书
- 代表作：
  - 《寒食帖》（被誉为"天下第三行书"）
  - 《祭黄几道文》
  - 《前赤壁赋》
- 主张"书初无意于佳乃佳"

**绘画**
- 擅长墨竹、枯木怪石
- 文人画的倡导者
- 主张"诗中有画，画中有诗"
- 传世作品罕见，多为后世摹本

## 🧠 思想特色

### 儒释道融合

| 思想来源 | 影响体现 |
|---------|----------|
| **儒学** | 忠君爱国，关心民瘼，积极入世 |
| **道家** | 超然物外，顺其自然，豁达洒脱 |
| **佛学** | 禅意深远，生死超然，"万法皆空" |

### 人生观

**核心思想**
- **豁达乐观**：面对逆境泰然处之
- **随遇而安**："此心安处是吾乡"
- **知行合一**：主张理论与实践结合

**名言佳句**
> "人生如逆旅，我亦是行人"

> "人有悲欢离合，月有阴晴圆缺，此事古难全"

> "不识庐山真面目，只缘身在此山中"

> "但愿人长久，千里共婵娟"

> "竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生"

## 🌍 历史影响

### 文学地位

| 领域 | 影响 |
|------|------|
| **词** | 开创豪放词派，打破婉约传统 |
| **诗** | 宋诗的代表人物，影响深远 |
| **文** | 唐宋八大家之一，文风影响后世 |
| **书法** | "宋四家"之首，影响千年 |
| **绘画** | 文人画理论奠基人 |

### 对后世的影响

**直接传承**
- 苏门四学士：黄庭坚、秦观、晁补之、张耒
- 影响南宋词风，辛弃疾等豪放词人

**跨时代影响**
- 明代：公安派、竟陵派继承其文风
- 清代：袁枚等性灵派推崇苏轼
- 现代：鲁迅、林语堂等名家推崇

**海外影响**
- 朝鲜、日本、越南等地影响深远
- 林语堂《苏东坡传》英文世界广为人知

### 名人评价

| 人物 | 评价 |
|------|------|
| **欧阳修** | "读苏轼书，不觉汗出，快哉快哉！" |
| **王安石** | 政见不同，但承认其才华 |
| **林语堂** | "苏东坡是个秉性难改的乐天派，是悲天悯人的道德家，是百姓的朋友" |
| **金圣叹** | 评点苏轼文章，叹为绝妙 |
