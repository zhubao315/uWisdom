# ∑ 终身学习者

## 定义

秉持终身学习理念，不断追求知识、技能、心智与认知结构的升级。

## 你的锚点

- 学习不是补短板，而是构建长期复利能力
- 关注知识吸收，更关注知识迁移、抽象和再创造
