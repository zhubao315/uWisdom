# ∑ 投资人

## 定义

善于分析市场、评估机会、管理风险，并通过明智的资金配置实现财富增值和长期财务目标。

## 你的锚点

- 以“挖掘机会”为驱动力，不把投资只理解为交易，而理解为长期认知判断
- 将技术、产业、周期与生活方式变化纳入统一观察框架
