# ∑ 投资人

标杆：查理·芒格 + 沈南鹏

## 定义

投资对我而言，不只是理财手段，而是一种认知活动：识别机会、判断赔率、管理风险、等待时间兑现价值。

我希望成为的投资人，不是追逐噪音的交易者，也不是靠运气吃饭的投机者，而是能够在不确定性中保持判断力、纪律性和长期主义的人。

投资的本质，是用资本为认知投票。

## 核心目标

- 识别真正值得长期持有的机会
- 用风险管理守住底线
- 用资产配置穿越周期
- 用复利而非频繁折腾，实现财富增长

## 我的投资原则

### 1. 机会优先，问题其次

成果来自挖掘机会，而不是沉迷于解决问题。投资也是如此，关键不只是规避错误，更是识别真正值得下注的方向。

### 2. 在能力圈内行动

只做自己能理解、能判断、能跟踪的投资。认知不到位的机会，再热闹也不碰。

### 3. 风险先于收益

先想清楚最差情况，再考虑上行空间。投资不是比谁更激进，而是比谁能活得更久。

### 4. 长期主义

短期价格会波动，长期价值会称重。真正的优势，不只是买得对，更是拿得住。

### 5. 组合管理

单点判断可能出错，但合理组合、持续复盘、动态调整，可以让整体决策越来越稳。

## 我的投资方法

1. **先看赛道**：先判断是不是长期值得关注的方向
2. **再看企业或资产质量**：关注商业模式、护城河、管理层、现金流
3. **然后做估值与赔率判断**：价值与价格之间的关系
4. **最后做仓位与节奏管理**：买多少、什么时候买、什么时候等

## 成长方向

- 建立更稳定的研究、记录与复盘机制
- 把投资从"看机会"升级为"有体系地管理机会"
- 增强对技术周期、产业变化和全球市场联动的理解
- 形成适合自己的长期投资框架

对我来说，投资人这个身份的意义，不只是实现财富增长，更是训练自己如何在复杂世界里做更清醒的判断。
