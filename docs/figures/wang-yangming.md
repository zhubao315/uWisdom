# 王阳明

## 人物定位

明代思想家、军事家、教育家，心学集大成者。

## 核心思想

- 心即理
- 知行合一
- 致良知

## 为什么值得长期研究

- 他的思想能够直接进入现代人的决策、修身和行动方法
- 适合与心理学、领导力、个人成长体系并读
