# 蒙田

## 人物定位

法国文艺复兴时期思想家，以《随笔集》奠定现代随笔与自我审视写作传统。

## 核心思想

- 我知道什么？
- 反独断
- 通过自我观察理解人

## 为什么值得长期研究

- 对研究写作、思辨表达和终身学习方法非常关键
- 适合与王阳明、现代心理学并置阅读
