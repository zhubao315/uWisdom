# 埃隆·马斯克

> **出生年**：1971 · **身份**：SpaceX / Tesla / xAI / Neuralink / The Boring Company 创始人或 CEO
> **领域**：航天、电动汽车、人工智能、脑机接口、社交媒体

## 人物定位

同时运营多家市值达数十亿美元公司的科技企业家。他的第一性原理思维方式、极致执行力和对人类未来的宏大愿景，为创业者和工程师提供了行动路线图。

## 核心工程方法

面对任何挑战，马斯克遵循五步工程流程：

1. **质疑现状** — 不接受"一直都是这么做的"
2. **抓住核心** — 找到问题的第一性原理
3. **简化并优化** — 删除不必要的部分，再优化
4. **快速反馈循环** — 缩短迭代周期
5. **自动化** — 但要在前四步之后进行

这就是 SpaceX 能以其他公司十倍的速度制造火箭的原因。

## 管理哲学

- **小而精悍的团队** — 不喜欢大团队，深知"帕金森定律"
- **身处一线** — 只有真正参与项目的人才能做出最佳决策
- **跳过层级** — 99% 的时间与一线工程师交流，而非管理层
- **消除瓶颈** — 一旦发现问题立即清除，不走冗长流程
- **重视尊重，而非取悦** — 激励别人达到最佳状态是馈赠，迁就是错误

## 核心思想

- **第一性原理** — 从物理本质出发思考，不依赖类比
- **北极星使命** — 确保人类能够持续存续，成为多星际文明
- **永不休息文化** — 用看似不可能的时间表驱动极致产出
- **了解自己** — 充分发挥长处，坦然接受缺点

## 对你的价值

- 第一性原理思维对架构设计有直接启发
- 五步工程流程可应用于系统设计与项目管理
- "质疑 → 简化 → 优化 → 自动化"的方法论通用性极强
