# 户外装备框架

所属领域：[户外大神]({{ '/knowledge-base/domains/outdoor-adventure.html' | relative_url }})

## 定义

户外装备的选型原则、搭配逻辑和轻量化策略，确保功能性和舒适性的平衡。

## 核心要点

### 选型原则

- 场景匹配：根据活动类型（日归 / 多日 / 高海拔）选择
- 重量 vs 耐久：轻量化不等于牺牲可靠性
- 三层法则：买预算内最好的鞋、背包和外套

### 核心装备

- 鞋：徒步鞋 / 越野跑鞋，防水 vs 透气的取舍
- 背包：合适的容量 + 良好的背负系统
- 衣物：三层穿衣法 + 防雨方案

### 轻量化策略

- 称重：记录每件装备的重量
- 一物多用：减少专一功能装备
- 分享：团队装备共享（帐篷、炊具）
