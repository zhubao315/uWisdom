# 游戏机制分析

所属领域：[游戏王者]({{ '/knowledge-base/domains/gaming.html' | relative_url }})

## 定义

分析游戏设计的核心机制（资源管理、博弈、成长曲线、反馈循环），并将游戏化思维应用于现实任务。

## 核心要点

### 核心机制

- **资源管理**：有限资源下的分配决策
- **博弈论**：多人互动中的策略选择
- **成长曲线**：能力提升的节奏与激励
- **反馈循环**：即时反馈驱动持续投入

### 游戏化应用

- 积分、徽章、排行榜（PBL）用于习惯养成
- 任务拆解：把大目标拆成可完成的"关卡"
- 进度可视化：让努力被"看见"

### 现实迁移

- 工作中的"成就系统"设计
- 学习中的"技能树"规划
- 产品设计的游戏化元素
