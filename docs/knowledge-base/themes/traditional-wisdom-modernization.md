# 传统智慧现代化

所属领域：[国学大师]({{ '/knowledge-base/domains/chinese-classics.html' | relative_url }})

## 定义

将古代思想（儒释道、西方古典哲学）用现代语言重新表达，并应用到当代生活、管理和决策中的方法。

## 核心要点

### 转译方法

- 保留核心思想，替换古代语境
- 用现代案例和比喻重新阐释
- 连接心理学、管理学等现代学科

### 应用场景

- 管理：王阳明"致良知" → 领导者的自我觉察
- 决策：苏轼"也无风雨也无晴" → 逆境中的情绪管理
- 修身：《大学》"格物致知" → 系统性学习方法

### 常见误区

- 生搬硬套：不分场景强行引用古语
- 过度简化：丢失原典的复杂性
- 脱离实践：停留在知识层面，不落实到行动
