# 文化行走路线

所属领域：[北京文化使者]({{ '/knowledge-base/domains/beijing-culture.html' | relative_url }})

## 定义

设计和记录基于城市文化的步行路线，将历史、建筑、人文和日常生活串联为可体验的空间叙事。

## 核心要点

### 路线设计

- 主题化：按历史时期、文化类型、建筑风格组织
- 节奏感：张弛有度，有重点有留白
- 实用性：标注交通、餐饮、休息点

### 记录方法

- GPS 轨迹 + 照片 + 文字笔记
- 按时间线组织：出发 → 途中观察 → 终点感悟
- 可复现：提供路线指引供他人复走

### 经典路线示例

- 中轴线：永定门 → 钟鼓楼，7.8 公里的城市脊梁
- 胡同漫游：南锣鼓巷 → 什刹海周边
- 798 艺术区 → 望京小街
