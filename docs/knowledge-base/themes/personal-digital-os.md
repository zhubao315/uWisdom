# 个人数字操作系统

所属领域：[数字人生]({{ '/knowledge-base/domains/digital-life.html' | relative_url }})

## 定义

构建个人数字操作系统的完整框架，覆盖知识管理、任务管理、信息流、自动化和数字资产管理。

## 核心模块

### 知识管理

- 笔记系统：Obsidian / Logseq / Notion
- 知识组织：Zettelkasten / PARA / 主题化
- 检索能力：全文搜索 + 标签 + 双向链接

### 任务管理

- 任务捕获：快速记录，减少认知负担
- 优先级排序：重要/紧急四象限
- 每日/每周复盘机制

### 信息流

- RSS 阅读器管理订阅源
- 稍后阅读（Readwise / Pocket）
- 信息去噪：减少低价值信息源

### 自动化

- 日常任务自动化（IFTTT / Zapier / n8n）
- 数据备份自动化
- AI 辅助：摘要生成、分类整理
