# 跑步周期训练

所属领域：[健身跑者]({{ '/knowledge-base/domains/fitness-running.html' | relative_url }})

## 定义

将跑步训练分为不同周期（基础期、强化期、竞赛期、恢复期），系统化提升体能和成绩。

## 核心要点

### 训练周期

- **基础期**（4-8 周）：有氧基础，低强度长距离，建立跑量
- **强化期**（4-6 周）：加入节奏跑、间歇跑，提升速度耐力
- **竞赛期**（2-4 周）：减量调整，保持状态，准备比赛
- **恢复期**（1-2 周）：低强度活动，身体恢复与适应

### 训练类型

- 轻松跑（Easy）：能说话的配速，占总跑量 70-80%
- 节奏跑（Tempo）：乳酸阈值配速，提升耐力
- 间歇跑（Intervals）：高强度短距离反复，提升速度
- 长距离（Long Run）：周末长跑，建立有氧耐力

### 数据监控

- 心率区间训练
- 配速与跑量趋势
- 疲劳恢复指标（静息心率、睡眠质量）
