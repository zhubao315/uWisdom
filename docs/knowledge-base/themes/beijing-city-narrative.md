# 城市文化叙事

所属领域：[北京文化使者]({{ '/knowledge-base/domains/beijing-culture.html' | relative_url }})

## 定义

以北京为样本，研究城市文化、空间变迁和人文故事，形成有深度的城市观察与叙事能力。

## 核心要点

### 叙事维度

- 历史层：城市规划演变、地标建筑、历史事件
- 空间层：胡同、四合院、商业区、新城区的空间逻辑
- 人物层：文化名人、市井人物、当代创作者
- 生活层：饮食、节气、方言、日常习惯

### 观察方法

- 城市漫步（City Walk）：有目的地在街区中行走
- 口述历史：与老居民交流，记录即将消失的记忆
- 对比观察：同一地点不同时间的变化

### 内容输出

- 图文结合：摄影 + 文字
- 主题系列：按胡同、按行业、按时代组织
- 线上分享与线下行走结合
