# 认知偏差清单

所属领域：[心理学专家]({{ '/knowledge-base/domains/psychology.html' | relative_url }})

## 定义

整理和研究影响判断与决策的常见认知偏差，建立可查阅的偏差清单，提升理性思考能力。

## 核心偏差

### 信息处理偏差

- **确认偏差**：只关注支持自己观点的信息
- **锚定效应**：过度依赖第一个获得的信息
- **可得性偏差**：容易想起的事被认为更常见

### 社会偏差

- **从众效应**：跟随多数人的选择
- **权威偏差**：过度信任权威人士的意见
- **光环效应**：因某一优点而全面肯定

### 决策偏差

- **损失厌恶**：损失的痛苦 > 等额收益的快乐
- **沉没成本谬误**：因已投入而继续错误决策
- **过度自信**：高估自己判断的准确性

### 应对策略

- 建立决策检查清单
- 引入外部视角（魔鬼代言人）
- 强制考虑反面证据
- 记录决策日志，定期复盘
