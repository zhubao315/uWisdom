# 市场周期观察

所属领域：[金融市场]({{ '/knowledge-base/domains/financial-markets.html' | relative_url }})

## 定义

观察和记录宏观经济周期、市场情绪变化和资产轮动规律，建立系统性的市场观察框架。

## 核心要点

### 周期识别

- 经济周期四阶段：复苏 → 扩张 → 滞胀 → 衰退
- 领先指标 vs 同步指标 vs 滞后指标
- 央行政策周期：加息 → 暂停 → 降息 → 扩表

### 情绪指标

- VIX 恐慌指数
- 信用利差变化
- 资金流向（股票基金、债券基金、货币基金）

### 资产轮动

- 股债跷跷板效应
- 大小盘风格切换
- 价值 vs 成长轮动
