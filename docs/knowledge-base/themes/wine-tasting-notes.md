# 品鉴笔记

所属领域：[葡萄酒大师]({{ '/knowledge-base/domains/wine.html' | relative_url }})

## 定义

葡萄酒品鉴笔记的撰写方法、记录框架和积累技巧，将感官体验转化为可比较、可追溯的文字记录。

## 核心要点

### 品鉴步骤

- **观色**：澄清度、颜色深浅、色调（柠檬/金/琥珀）
- **闻香**：一类香气（果香/花香）、二类香气（发酵）、三类香气（陈年）
- **品味**：酸度、单宁、酒体、余味长度、平衡度

### 笔记框架

- 基本信息：酒庄、产区、年份、品种
- 感官描述：颜色 + 香气 + 口感
- 个人评价：喜好度（1-5）、性价比、是否回购

### 积累技巧

- 每次品鉴记录，不要依赖记忆
- 定期回顾旧笔记，感知自己的味觉变化
- 横向对比：同品种不同产区、同年份不同酒庄
