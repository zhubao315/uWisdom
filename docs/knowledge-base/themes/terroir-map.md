# 风土地图

所属领域：[葡萄酒大师]({{ '/knowledge-base/domains/wine.html' | relative_url }})

## 定义

绘制世界主要葡萄酒产区的风土（Terroir）地图，理解气候、土壤、地形和人文如何影响葡萄酒风格。

## 核心要点

### 旧世界

- **法国**：波尔多（混酿）、勃艮第（黑皮诺/霞多丽）、香槟、罗纳河谷
- **意大利**：皮埃蒙特（Barolo）、托斯卡纳（Chianti/Brunello）
- **西班牙**：里奥哈、普里奥拉托

### 新世界

- **美国**：纳帕谷（赤霞珠）、索诺玛、俄勒冈
- **澳大利亚**：巴罗萨谷（西拉）、猎人谷
- **智利 / 阿根廷**：门多萨（马尔贝克）、迈波谷

### 风土要素

- 气候：大陆性 vs 海洋性 vs 地中海
- 土壤：石灰岩、砾石、火山土、板岩
- 人文：酿酒传统、法规、创新
