# 视觉叙事

所属领域：[摄影家]({{ '/knowledge-base/domains/photography.html' | relative_url }})

## 定义

用影像（照片、视频）讲述故事的能力训练，从观察、构图到编辑和输出的完整方法论。

## 核心要点

### 观察力训练

- 光线意识：观察不同时间、天气下的光线变化
- 细节敏感：关注纹理、色彩、形状的组合
- 人文观察：人物表情、肢体语言、空间关系

### 构图方法

- 三分法、引导线、框架构图
- 减法原则：画面中只保留必要的元素
- 情绪传达：明暗、色彩、虚实如何影响感受

### 系列作品

- 选题策划：确定主题、目标受众、表现形式
- 编辑逻辑：照片之间的节奏和叙事连贯
- 输出形式：画册、展览、社交媒体
