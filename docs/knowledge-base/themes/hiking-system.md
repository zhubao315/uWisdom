# 徒步系统

所属领域：[户外大神]({{ '/knowledge-base/domains/outdoor-adventure.html' | relative_url }})

## 定义

构建系统化的徒步活动管理体系，覆盖路线规划、装备管理、体能准备和安全预案。

## 核心要点

### 路线规划

- 路线评估：距离、爬升、技术难度、补给点
- 天气窗口：出发前 3 天开始关注
- 预案 B：撤退路线和备用方案

### 装备系统

- 分层穿衣：排汗层 + 保暖层 + 防护层
- 背包系统：根据天数选择容量（日用 20-30L / 多日 40-60L）
- 导航工具：离线地图、GPS 设备、纸质地图备用

### 安全体系

- 结伴出行或告知他人行程
- 急救包和基本急救知识
- 通讯预案：信号盲区的应对
