# 经典阅读体系

所属领域：[国学大师]({{ '/knowledge-base/domains/chinese-classics.html' | relative_url }})

## 定义

系统性阅读中国传统经典和西方哲学经典的方法论，从选书、阅读、笔记到内化的完整流程。

## 核心要点

### 选书原则

- 从源头读起：先读原典，再读注疏
- 中西对照：同一问题的不同文化视角
- 由浅入深：入门选本 → 全本 → 注疏本

### 阅读方法

- 精读 vs 泛读：核心经典精读，扩展阅读泛读
- 主题阅读：围绕一个主题交叉阅读多本经典
- 费曼式理解：用自己的话复述核心思想

### 笔记与沉淀

- 抄录 + 批注 + 个人理解
- 按主题整理，而非按书整理
- 定期回顾，内化为自己的判断框架

### 推荐书单

- 儒家：《论语》《大学》《中庸》《孟子》
- 道家：《道德经》《庄子》
- 西方：《沉思录》（马可·奥勒留）、《随笔集》（蒙田）
