# 驾驭工程 (Knowledge Engineering)

## 1. 核心定义

驾驭工程是一套旨在提升知识“工程化可读性”的准则体系。它强调知识不仅是为人书写的，更是为 AI Agent 设计的。

## 2. 核心规则/方法

- **模块化**：知识应拆分为原子化的单元。
- **结构化**：必须具备机器可解析的元数据（Front-matter）。
- **版本化**：知识的演进应可追溯。

## 3. 验证示例

uWisdom 项目本身即是驾驭工程的第一个大规模实践案例。

## 4. 应用场景

- 个人知识库建设
- 企业智能大脑构建
- RAG 优化
