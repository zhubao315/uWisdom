# 商品交易

## 领域定义

关注能源、金属、农产品等商品市场的供需、周期、政策与交易机会。增强对现实经济与周期波动的敏感度，补足股票之外的机会识别能力。

## 对你的价值

- 商品是对通胀、地缘、供需变化最直接的资产类别
- 帮助建立跨资产类别的配置能力
- 黄金等贵金属在组合中承担避险与对冲功能

## 核心关注方向

### 供需分析

- 大宗商品的供给约束与需求驱动因素
- 库存周期与价格弹性
- 季节性因素与气候影响

### 周期与宏观

- 商品超级周期的历史规律
- 通胀与商品价格的关系
- 美元走势与商品定价

### 黄金与贵金属

- 黄金的避险属性与货币属性
- 央行购金趋势与长期供需
- 黄金在资产配置中的角色

### 交易框架

- 周期交易的基本逻辑
- 事件驱动与趋势跟踪
- 仓位管理与风险控制

## 典型 Project

- 黄金投资研究与跟踪
- 大宗商品周期分析
- 供需事件跟踪框架
- 周期交易策略搭建
