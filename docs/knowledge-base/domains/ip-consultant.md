# 知识产权顾问

## 领域定义

围绕专利、版权、软件成果、技术方案边界与知识资产保护进行判断与咨询。让技术成果可以被保护、转让、复用和商业化，把"做出来"升级为"能被确权和沉淀"。

## 对你的价值

- 在技术工作中理解知识产权边界，避免合规风险
- 帮助团队在开源使用、技术合作与商业保护之间找到平衡
- 是架构师从"纯技术"走向"技术 + 法务 + 商业"复合能力的重要维度

## 核心关注方向

### 软件成果保护

- 软件著作权登记与保护策略
- 核心算法与架构的专利化路径
- 商业秘密保护与信息分级

### 开源合规

- 开源许可证选择与风险评估
- 开源组件的合规审查流程
- 开源与商业的边界管理

### 专利策略

- 技术创新的专利挖掘与布局
- 专利质量 vs 数量的取舍
- 专利的防御性与进攻性使用

## 典型 Project

- 技术成果梳理与专利机会识别
- 软件资产归档与保护
- 核心技术成果的专利布局
- 开源合规审查与治理
