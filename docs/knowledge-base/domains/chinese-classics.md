# 国学大师

## 领域定义

围绕中国传统经典、思想源流、历史智慧与修身方法的长期学习与理解。从经典中寻找稳定的价值锚点，把传统智慧转译到现代语境中。

## 对你的价值

- 补充现代技术路径之外的深层认知结构
- 提升判断、修养与表达的文化厚度
- 国学提供价值观、人生观和长期尺度
- 与心理学形成互补：东方修养 + 西方认知科学

## 核心关注方向

### 经典阅读

- 儒家经典（论语、大学、中庸、孟子）
- 道家思想（道德经、庄子）
- 佛学入门与现代转译
- 史记、资治通鉴等历史智慧

### 思想脉络

- 先秦诸子百家的争鸣与互补
- 宋明理学与心学（朱熹、王阳明）
- 近代思想转型与现代价值重估

### 知行合一

- 修身方法与日常实践
- 传统智慧在现代生活中的应用
- 以经典指导决策、关系与自我管理

### 现代转译

- 把古典思想用现代语言重新表达
- 跨文化比较与融合
- 传统智慧在商业、管理、科技中的映照

## 标杆人物

### 王阳明

- 心学的核心："致良知"与"知行合一"
- 在困境中建立内心秩序的实践智慧
- 提醒我：真正的认知必须落实为行动

### 苏轼

- 跨越文学、政治、艺术与生活美学的复合型人物
- 在逆境中的精神韧性与旷达
- 提醒我：审美、思想与烟火气可以统一

## 典型 Project

- 经典阅读计划与笔记
- 思想脉络整理
- 传统智慧现代化表达
- 王阳明心学实践研究
