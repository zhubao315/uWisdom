# 户外大神

## 领域定义

围绕徒步、露营、自然探索和户外技能建设展开的生活实践领域。保持身体活力，也保持与自然的连接。

## 对你的价值

- 平衡高强度脑力工作，为身心提供恢复与更新
- 建立身体、自然与精神状态的连接
- 户外是生活艺术家的"在场"体验：用身体感受空间、气候与地形

## 核心关注方向

### 徒步与穿越

- 路线规划与难度评估
- 不同地形的行走技巧
- 多日穿越的装备与后勤

### 装备系统

- 户外装备的选型与搭配
- 轻量化与功能性的平衡
- 不同季节与场景的装备方案

### 自然观察

- 山野、溪流、植被、天气的观察力
- 户外摄影与自然记录
- 与自然相处的心境与节奏

### 安全与技能

- 户外安全知识（导航、急救、天气判断）
- 露营技能与野外生存基础
- 团队协作与风险管理

## 典型 Project

- 徒步路线计划与执行
- 户外装备系统化整理
- 自然体验记录与输出
- 季节性户外活动规划
