# 健身跑者

## 领域定义

通过跑步、力量训练与体能管理维持长期健康、纪律与精力水平。身体是一切体验的基础，也是一切长期认知输出的底层平台。

## 对你的价值

- 身体是长期认知输出的底层平台
- 稳定训练是提升意志与节律管理的核心方式
- 生活艺术不只在精神层面，更要落实到身体状态和精力管理上

## 核心关注方向

### 跑步训练

- 周期化训练计划（基础期、强化期、竞赛期、恢复期）
- 配速、心率与跑量管理
- 不同跑步类型（轻松跑、节奏跑、间歇跑、长距离）

### 力量训练

- 核心力量与稳定性训练
- 跑步专项力量（臀腿、核心）
- 上下肢均衡发展

### 体能管理

- 睡眠、营养与恢复
- 训练负荷监控与过度训练预防
- 身体数据追踪与复盘

### 长期可持续

- 避免伤病的训练策略
- 年龄与训练方式的适配
- 把健康当作长期工程，而非短期冲刺

## 典型 Project

- 周训练计划制定与执行
- 跑步数据复盘与分析
- 体能提升周期设计
- 跑步与力量训练的结合方案
