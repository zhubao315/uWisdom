# 人工智能专家

## 领域定义

围绕 AIGC、LLM、Agent、提示工程、RAG 与 AI 应用落地进行研究和实践。这是我当前职业的主航道，也是 uWisdom 未来 Agent 化的核心驱动领域。

## 对你的价值

- 这是你当前职业主航道（2024 至今）
- 将 LLM、知识库、Agent、工作流编排等能力纳入架构设计范畴
- 从"系统架构"走向"知识架构 + 智能架构"

## 核心关注方向

### LLM 应用架构

- 大语言模型的工程化落地路径
- Prompt 体系设计与管理
- 上下文窗口管理与成本优化

### Agent 工作流设计

- 单 Agent 与多 Agent 协作模式
- 工具调用、记忆管理与任务编排
- Agent 与现有系统的集成架构

### RAG 与知识系统

- 检索增强生成的设计与优化
- 向量数据库选型与索引策略
- 知识库的构建、维护与质量保障

### 企业级 AI 落地

- AI 能力平台化与服务化
- 模型评估、安全与合规
- 从 PoC 到生产的工程化路径

## 典型 Project

- LLM 应用架构设计
- Agent 工作流搭建
- 企业级 AI 落地方案
- 个人知识系统 Agent 化（uWisdom）
