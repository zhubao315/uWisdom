# 产品经理

## 领域定义

以用户需求为导向，连接业务目标与技术实现，围绕用户价值、产品边界、需求优先级与落地路径进行设计与取舍。

## 对你的价值

- 产品意识帮助你理解用户与业务价值，而非只关注技术实现
- 保证技术建设服务真正的价值目标
- 把机会判断转化为清晰的产品结构

## 核心关注方向

### 需求分析与定义

- 用户研究与需求挖掘
- 需求优先级排序与取舍
- PRD 撰写与需求沟通

### 产品设计

- 信息架构与交互设计
- 用户体验与可用性
- 数据驱动的产品迭代

### 技术产品化

- 把技术能力转化为产品功能
- AI 能力的产品化包装
- 开发者产品与 API 产品设计

## 典型 Project

- uWisdom PRD 体系设计
- Agent 产品化方案
- 商业案例库规划
- AI 能力产品化方案
