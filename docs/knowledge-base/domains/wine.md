# 葡萄酒大师

## 领域定义

围绕葡萄酒文化、品鉴方法、风土知识与社交表达展开的审美型学习领域。提升味觉、审美和社交场景表达能力，把生活体验升级为可描述、可比较、可分享的知识。

## 对你的价值

- 训练感知力、审美力和表达力
- 葡萄酒是连接风土、文化、感官与社交的综合载体
- 与摄影形成互补：一个训练视觉审美，一个训练味觉审美

## 核心关注方向

### 品鉴方法

- 葡萄酒品鉴的系统方法（观色、闻香、品味）
- 主要葡萄品种的特征识别
- 盲品技巧与品鉴笔记撰写

### 风土知识

- 世界主要产区的风土特征
- 法国、意大利、西班牙、新世界产区对比
- 气候、土壤与酿造工艺对风格的影响

### 餐酒搭配

- 餐酒搭配的基本原则
- 中餐与葡萄酒的搭配实践
- 不同场景的选酒策略

### 文化与社交

- 葡萄酒的历史与文化背景
- 酒庄拜访与产区游历
- 葡萄酒社交场景的得体表达

## 典型 Project

- 葡萄酒品鉴笔记库建设
- 产区知识地图绘制
- 酒餐搭配清单与实践
- 风土地图研究
