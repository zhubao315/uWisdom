# 金融市场

## 领域定义

研究股票、债券、基金、指数及其背后的宏观环境、产业趋势与资金行为。建立系统性的市场观察框架，训练风险收益判断能力。

## 对你的价值

- 帮助你在不确定性中保持判断力、纪律性和长期主义
- 投资的本质，是用资本为认知投票
- 与架构师视角互补：用技术判断辅助投资决策

## 核心关注方向

### 宏观与周期

- 经济周期识别（繁荣、衰退、萧条、复苏）
- 货币政策与流动性分析
- 全球市场联动与地缘影响

### 行业与赛道

- 长期趋势赛道筛选
- 行业竞争格局与护城河分析
- 产业拐点与技术代际变化

### 估值与风险管理

- 企业价值评估方法（DCF、相对估值等）
- 安全边际原则
- 组合管理与仓位控制

### 价值投资方法

- 在能力圈内行动：只做自己能理解的投资
- 风险先于收益：先想清楚最差情况
- 长期主义：短期价格波动，长期价值称重

## 典型 Project

- 行业趋势跟踪与研究
- 投资组合复盘与优化
- 市场周期观察与记录
- 价值投资框架搭建
