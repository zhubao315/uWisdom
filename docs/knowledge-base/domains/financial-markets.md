# 金融市场

## 领域定义

研究股票、债券、基金、指数及其背后的宏观环境、产业趋势与资金行为。建立系统性的市场观察框架，训练风险收益判断能力。

**核心命题**：在不确定性中寻找可判断的机会，用纪律约束情绪，用长期视角对抗短期波动。

**与技术的互补**：
- 技术判断 → 筛选具有技术护城河的公司
- 架构思维 → 评估公司组织结构和执行力
- 系统思维 → 理解产业链和生态位

## 对你的价值

### 认知价值

1. **决策框架**：在不确定性中保持判断力的方法论
2. **风险意识**：理解风险、度量风险、管理风险
3. **长期视角**：抵抗短期波动的心理建设
4. **跨学科思维**：将工程思维应用到投资领域

### 能力构建

- 宏观经济分析框架
- 行业研究与赛道筛选
- 企业估值与定价
- 组合管理与风险控制

## 核心关注方向

### 1. 宏观与周期

#### 1.1 经济周期框架

**美林时钟**（Merrill Lynch Clock）：

```
        通胀下行            通胀上行
           │                   │
    ┌──────┴──────┐    ┌──────┴──────┐
    │    衰退      │    │    滞胀     │
    │   债券+现金  │    │   现金为主   │
    └──────┬──────┘    └──────┬──────┘
           │                   │
    ┌──────┴──────┐    ┌──────┴──────┐
    │    复苏      │    │    繁荣     │
    │   股票为主   │    │   商品为主   │
    └──────┬──────┘    └──────┬──────┘
           │                   │
        增长上行            增长下行
```

**周期定位信号**：

| 阶段 | 领先指标 | 同步指标 | 滞后指标 |
|------|----------|----------|----------|
| 衰退 | 国债收益率曲线倒挂 | GDP 负增长 | 失业率上升 |
| 复苏 | PMI 回升 | 工业增加值 | 通胀回升 |
| 繁荣 | 产能利用率高 | GDP 增长 | 通胀上行 |
| 滞胀 | 油价冲击 | 失业率低 | 政策收紧 |

### 2. 行业与赛道

#### 2.1 赛道筛选框架

**长期趋势赛道特征**：

| 特征 | 说明 | 示例 |
|------|------|------|
| **刚需性** | 需求稳定，不受经济周期影响 | 食品饮料、医疗 |
| **成长性** | 市场规模持续扩大 | 新能源、AI |
| **壁垒性** | 先发优势或护城河 | 半导体、云服务 |
| **定价权** | 能够转嫁成本 | 奢侈品、高端制造 |

**产业生命周期定位**：

```
渗透率
  ▲
  │    ╭────────────╮
  │   ╱              ╲
  │  ╱                ╲
  │ ╱                  ╲
  │╱                    ╲
  └──────────────────────────► 时间
   初创  成长  成熟  衰退
```

- **初创期**：渗透率 < 5%，技术路线不确定
- **成长期**：渗透率 5%-50%，赢家通吃
- **成熟期**：渗透率 > 50%，竞争激烈
- **衰退期**：替代品出现，市场萎缩

#### 2.2 竞争格局分析

**波特五力模型**：

1. **行业内竞争**：市场集中度、主要玩家、竞争强度
2. **潜在进入者**：进入壁垒、规模经济、品牌认知
3. **替代品威胁**：替代品性价比、用户转换成本
4. **供应商议价**：集中度、替代供应、可得性
5. **买家议价**：集中度、信息透明度、转换成本

**护城河类型**：

| 类型 | 说明 | 代表公司 |
|------|------|----------|
| 无形资产 | 品牌、专利、特许经营权 | 可口可乐、茅台 |
| 转换成本 | 用户习惯、锁定效应 | Adobe、Salesforce |
| 网络效应 | 用户越多价值越高 | 微信、AWS |
| 成本优势 | 规模经济、地理位置 | 拼多多、亚马逊 |

#### 2.3 产业拐点识别

**技术代际变化信号**：
- 现有技术达到物理极限
- 新技术成本曲线快速下降
- 政策大力支持
- 领先厂商开始转型

**行业拐点案例**：

| 行业 | 拐点 | 信号 |
|------|------|------|
| 智能手机 | 2010 年代 | iPhone 重新定义手机 |
| 云计算 | 2015 年代 | AWS 盈利证明模式 |
| 新能源车 | 2020 年代 | 续航突破 + 政策补贴 |
| AI | 2023 年代 | ChatGPT 破圈 |

### 3. 估值与风险管理

#### 3.1 估值方法体系

**绝对估值法（DCF）**：

```python
def dcf_valuation(
    free_cash_flows: list,  # 未来各年自由现金流
    terminal_growth: float,  # 永续增长率
    discount_rate: float,   # 折现率 (WACC)
    terminal_multiple: float = None  # 终值倍数
) -> dict:
    """
    DCF 估值核心逻辑：
    企业价值 = Σ(FCF_t / (1+r)^t) + TV / (1+r)^n
    """
    # 1. 计算各年折现值
    present_values = []
    for t, fcf in enumerate(free_cash_flows, 1):
        pv = fcf / ((1 + discount_rate) ** t)
        present_values.append(pv)
    
    # 2. 计算终值
    if terminal_multiple:
        terminal_value = free_cash_flows[-1] * terminal_multiple
    else:
        terminal_value = free_cash_flows[-1] * (1 + terminal_growth) / (discount_rate - terminal_growth)
    
    # 3. 折现终值
    pv_terminal = terminal_value / ((1 + discount_rate) ** len(free_cash_flows))
    
    # 4. 计算企业价值
    enterprise_value = sum(present_values) + pv_terminal
    
    return {
        "enterprise_value": enterprise_value,
        "pv_terminal": pv_terminal,
        "pv_terminal_ratio": pv_terminal / enterprise_value,  # 终值占比，过高需警惕
        "sensitivity": {
            f"wacc_{delta}": dcf_valuation(free_cash_flows, terminal_growth, discount_rate + delta, terminal_multiple)
            for delta in [-0.01, 0.01]
        }
    }
```

**相对估值法**：

| 指标 | 适用场景 | 参考基准 |
|------|----------|----------|
| P/E（市盈率） | 盈利稳定的成熟企业 | 历史均值、行业均值 |
| P/B（市净率） | 金融、周期性行业 | 1x 净资产 |
| P/S（市销率） | 成长早期、亏损企业 | 行业均值 |
| EV/EBITDA | 跨行业比较 | 行业龙头 |
| PEG | 成长股（G > 20%） | < 1 有价值 |

#### 3.2 安全边际

**安全边际三层含义**：

1. **价格保护**：买入价格 < 内在价值，留出容错空间
2. **能力圈保护**：只投资自己理解的公司
3. **周期保护**：在市场恐慌时买入，而非理性繁荣时

**安全边际计算**：

```python
def calculate_margin_of_safety(
    intrinsic_value: float,  # 内在价值
    current_price: float,    # 当前价格
    confidence: float = 0.8  # 估值置信度
) -> dict:
    """
    安全边际 = (内在价值 - 当前价格) / 内在价值
    """
    mos = (intrinsic_value - current_price) / intrinsic_value
    
    # 调整置信度
    adjusted_mos = mos * confidence
    
    return {
        "raw_mos": mos,
        "adjusted_mos": adjusted_mos,
        "recommendation": (
            "强烈买入" if adjusted_mos > 0.4 else
            "买入" if adjusted_mos > 0.2 else
            "持有" if adjusted_mos > 0 else
            "卖出"
        )
    }
```

#### 3.3 风险管理原则

**仓位管理原则**：

| 原则 | 说明 |
|------|------|
| **单票上限** | 单个股票仓位 ≤ 20% |
| **行业分散** | 单个行业仓位 ≤ 30% |
| **现金仓位** | 保留 10-20% 现金应对机会 |
| **逆势加仓** | 下跌时分批建仓，而非一次性买入 |

**止损纪律**：

```python
def should_stop_loss(
    entry_price: float,
    current_price: float,
    stop_loss_ratio: float = 0.15,  # 默认止损 15%
    trailing_stop: bool = True,
    highest_price: float = None
) -> dict:
    """
    止损决策
    """
    if trailing_stop and highest_price:
        stop_price = highest_price * (1 - stop_loss_ratio)
        is_stopped = current_price <= stop_price
        stop_type = "追踪止损"
    else:
        stop_price = entry_price * (1 - stop_loss_ratio)
        is_stopped = current_price <= stop_price
        stop_type = "固定止损"
    
    return {
        "is_stopped": is_stopped,
        "stop_price": stop_price,
        "stop_type": stop_type,
        "loss_ratio": (entry_price - current_price) / entry_price,
        "action": "止损" if is_stopped else "持有"
    }
```

### 4. 价值投资方法论

#### 4.1 价值投资核心原则

**巴菲特投资哲学**：

| 原则 | 具体含义 |
|------|----------|
| **竞争优势** | 选择有护城河的公司 |
| **管理层** | 诚信、有能力的管理层 |
| **价格合理** | 买入价格有安全边际 |
| **长期持有** | 伴随优秀企业成长 |

**能力圈原则**：

> "知道你不知道什么比知道你知道什么更重要。" — 芒格

**能力圈评估框架**：

```
我能理解的公司特征：
├── 主营业务简单、单一
├── 商业模式清晰、可预测
├── 行业竞争格局稳定
├── 有可获取的信息来源
└── 能用简单的语言描述
```

#### 4.2 研究流程

**投资研究五步法**：

```
Step 1: 筛选
├── 行业筛选：长期趋势、竞争格局
├── 财务筛选：盈利能力、现金流、负债率
└── 估值筛选：PE、PB、EV/EBITDA

Step 2: 初步分析
├── 业务理解：做什么、怎么做
├── 商业模式：怎么赚钱
└── 竞争地位：护城河在哪里

Step 3: 深度研究
├── 财务分析：3-5 年历史数据
├── 管理层访谈：战略、执行力
├── 竞争对手比较：优劣势分析
└── 行业专家咨询：验证判断

Step 4: 估值
├── 绝对估值：DCF 三种情景
├── 相对估值：同业比较
└── 锚定价格：极端乐观/悲观

Step 5: 决策
├── 综合判断：机会 vs 风险
├── 仓位决策：配置比例
├── 买入时机：分批建仓
└── 跟踪计划：监控指标
```

#### 4.3 卖出决策

**卖出三原则**：

| 情况 | 说明 | 操作 |
|------|------|------|
| **买错了** | 投资逻辑被证伪 | 立即卖出，不纠结 |
| **太贵了** | 估值远超内在价值 | 分批卖出，持有现金 |
| **更好的机会** | 发现更高性价比标的 | 仓位调整 |

**不卖出的情况**：
- 股价短期下跌，但基本面没变
- 对公司长期有信心
- 没有更好的替代机会
