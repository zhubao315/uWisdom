# 云计算与云原生

## 领域定义

围绕容器、Kubernetes、平台工程、开发者平台（IDP）等方向，提升系统的弹性、交付效率与治理能力。

## 对你的价值

- 云原生能力帮助你设计可扩展、可交付的现代系统
- 是架构师角色的核心技术底盘之一
- 与 AI 原生系统形成互补：底层基础设施 + 上层智能能力

## 核心关注方向

### 容器与编排

- Docker 容器化实践与镜像优化
- Kubernetes 集群管理与调度策略
- 服务网格（Istio / Linkerd）与流量治理

### 平台工程

- 内部开发者平台（IDP）设计
- 平台化思维：抽象通用能力，降低业务团队门槛
- DevOps 工具链整合与自动化

### 云架构设计

- 多云与混合云架构策略
- 成本优化与资源治理
- 可观测性体系建设（日志、指标、链路追踪）

### 现代交付体系

- CI/CD 流水线设计
- GitOps 与声明式部署
- 渐进式发布与回滚策略

## 典型 Project

- 容器化改造与迁移
- K8s 平台建设与治理
- 开发者平台（IDP）规划
- 云原生治理框架
