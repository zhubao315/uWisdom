# 心理学专家

## 领域定义

研究认知、情绪、行为、关系与成长机制，提升对人和自我的理解能力。帮助更好理解人性、行为、关系与自我。

## 对你的价值

- 优化决策、协作与自我迭代
- 为投资、产品、管理和生活实践提供心智基础
- 与国学形成互补：东方修养 + 西方认知科学
- 心理学帮助你"看清人，也看清自己"

## 核心关注方向

### 认知与决策

- 认知偏差清单与识别方法
- 决策心理学与理性思考
- 系统 1 vs 系统 2 思维（快与慢）

### 情绪与自我

- 情绪识别与调节策略
- 自我认知与自我效能
- 压力管理与心理韧性

### 关系与沟通

- 人际关系的心理学原理
- 高效沟通与影响力
- 冲突管理与协商

### 成长与习惯

- 习惯养成的行为科学
- 内在动机与自我驱动
- 刻意练习与反馈机制

## 典型 Project

- 认知偏差清单与案例库
- 情绪管理策略与实践
- 高效协作心理模型
- 习惯养成实验
