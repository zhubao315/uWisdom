# 北京文化使者

## 领域定义

围绕北京历史、人文、城市空间、在地文化与当代生活表达进行探索与传播。把城市体验升级为文化叙事能力，建立地方文化与个人表达的连接。

## 对你的价值

- 北京是你的生活主场，值得深入了解和持续探索
- 城市文化是连接历史、空间、人物与日常生活的最佳载体
- 让学习不只停留在纸面，而能进入真实场景与表达

## 核心关注方向

### 历史与空间

- 北京城的规划演变（元明清至今）
- 中轴线、胡同、四合院的空间逻辑
- 历史地标与当代城市更新

### 文化与人物

- 北京文化圈的历史人物与当代传承
- 京味文化、市井文化与精英文化的互动
- 老北京生活方式的现代延续

### 城市观察

- 城市更新中的文化保护与商业开发
- 社区生活、街巷肌理与公共空间
- 城市漫步（City Walk）与文化行走

### 内容输出

- 城市文化摄影与视觉叙事
- 文化行走路线设计
- 北京故事的当代书写

## 典型 Project

- 北京文化地图绘制
- 胡同与城市记忆专题
- 文化行走路线设计
- 城市观察内容输出
