# 开源发烧友

## 领域定义

通过开源项目学习、参与、复用与共建，深入理解开源生态、社区协作机制、许可证体系与商业模式，以开源思维指导技术选型、系统设计与团队协作。

## 对你的价值

- 连接全球最佳实践，让个人积累进入公共知识生态
- 开源视角帮助你理解生态、标准与协作机制
- 在 AI 时代，开源是技术能力的核心来源之一（模型、框架、工具链）

## 核心关注方向

### 开源生态理解

- 主流开源项目的技术架构与演进路径
- 开源社区的治理模式与协作机制
- 开源基金会（CNCF、Apache、Linux Foundation）的运作方式

### 开源选型与评估

- 开源项目的成熟度评估框架
- 社区活跃度、维护质量与长期可持续性
- 开源与商业方案的对比决策

### 开源协作

- 参与开源社区的方式与最佳实践
- 内部项目的开源化路径
- 开源文化在团队中的落地

## 典型 Project

- 开源知识库建设（uWisdom）
- 开源工具评估与选型
- 社区协作流程沉淀
- 内部工具开源化
