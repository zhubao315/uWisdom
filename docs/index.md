## 当前强化方向

- 导航重构：按知识百科、研究、人物、工具组织站点
- 搜索优化：支持问题输入与结构过滤
- Agent 入口：条目页一键复制链接供 Agent 学习
- 知识图谱：自动生成 Mermaid 条目关系图
- 每日精选：构建时自动产出“今日知识精华”

## 方法锚点

> 设定我关注的领域，并不断探寻“领域”路径，保持正确迭代并跟随时代步伐。

> 成果源于挖掘机会，而非解决问题。
