# 任务计划索引

本目录存放 uWisdom 项目的任务计划，基于 Superpowers `writing-plans` 方法论。

## 计划结构

每个任务计划包含：

- **背景** - 为什么做这件事
- **目标** - 具体可衡量的完成标准
- **详细任务分解** - 2-5 分钟粒度的子任务
- **风险评估** - 潜在风险和应对方案
- **检查点** - 设计评审、代码审查、测试验证

## 工作流

```
1. 需求澄清 → brainstorming
   ↓
2. 编写计划 → writing-plans
   ↓ 用户确认
3. 执行任务 → test-driven-development
   ↓
4. 代码审查 → requesting-code-review
   ↓
5. 合并代码 → finishing-a-development-branch
```
