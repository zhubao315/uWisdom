# 个人知识操作系统

## 研究问题

如何把零散笔记升级为具备搜索、版本、Agent 接入与商业复用能力的知识系统？

## 研究框架

- 知识建模：领域、主题、标签、版本
- 检索设计：结构搜索、语义搜索、引用回链
- Agent 接入：复制链接、上下文装配、调用边界
- 发布机制：GitHub、Pages、自动化构建
