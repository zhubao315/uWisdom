# 个人知识操作系统

## 研究背景

个人知识管理正从"记笔记"向"构建知识操作系统"演进。核心问题不再是"如何记录"，而是"如何让知识持续演进、被检索、被调用"。

## 研究方向

### 知识组织

- 从文件夹到图谱：知识的关联性组织
- Zettelkasten、PARA、CODE 等方法论对比
- 元数据驱动的知识发现

### 检索能力

- 全文搜索 vs 语义搜索 vs 混合搜索
- 向量数据库在个人知识库中的应用
- 自然语言提问 → 知识条目匹配

### Agent 集成

- 个人知识库作为 Agent 的长期记忆
- 知识条目的机器可读格式设计
- Agent 自动关联和推荐相关知识

### 演进机制

- 知识的版本管理和生命周期
- 自动化更新提醒（过时内容检测）
- 复习间隔与知识强化
