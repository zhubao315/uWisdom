# OpenClaw 智能体生态

## 研究问题

OpenClaw 作为智能体平台与工具链，如何形成从知识底座、工具接入、工作流编排到生态协作的完整能力体系？

## 研究框架

- 平台层：模型接入、上下文管理、工具调用、权限边界
- 工具层：知识库、检索、工作流、MCP/插件能力
- 协同层：单 Agent、多 Agent、角色分工、链式调用
- 生态层：开源社区、技能市场、场景模板、企业接入

## 研究价值

- 帮助理解智能体生态而不是单点 Agent
- 为 uWisdom 的 Agent 可调用设计提供现实参照
- 支撑未来从个人知识系统走向智能体平台协作
