## 🎯 技能定位

**Agent Browser** 让 AI Agent 能够：
1. **实时搜索**：查询最新信息、趋势、数据
2. **网页浏览**：访问特定网站、获取内容
3. **信息验证**：交叉验证信息的准确性
4. **知识补充**：从外部源获取新知识
